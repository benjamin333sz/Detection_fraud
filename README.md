# Detection_fraude

Le but du projet est d'à partir d'un dataset, de retrouver les valeurs aberrantes grâce à des modèles de Machine Learning. Au cours de ce projet, nous avons ainsi plusieurs types de modèles z_clot, erreur relatif, Isolation Forest et un modèle from_scratch qui utilise une courbe Gaussienne. Nous avons ainsi pu voir l'efficacité ou non de ces modèles. (voir rapport world).

Ce GitHub s'articule de la manière suivante :
* Le fichier python Projet.py contient les modèles z_clot, erreur relatif et Isolation Forest.
* Le fichier model_from_scratch.py contient quant à lui un modèle fait à la main en utilisant une courbe gaussienne.
