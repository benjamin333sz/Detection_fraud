# Detection_fraude

Le but du projet est de retrouver à partir d'un dataset,  les valeurs aberrantes grâce à des modèles de Machine Learning. Au cours de ce projet, nous avons utilisé plusieurs types de modèles z_clot, erreur relatif, Isolation Forest et un modèle from_scratch qui utilise une courbe Gaussienne. Nous avons ainsi pu voir l'efficacité ou non de ces modèles. (voir rapport word).

Ce projet s'articule de la manière suivante :
* Le fichier python Projet.py contient les modèles z_clot, erreur relatif et Isolation Forest.
* Le fichier model_from_scratch.py contient quant à lui un modèle fait à la main en utilisant une courbe gaussienne.
* Vous pouvez voir une interface graphique dans le fichier interface.py
